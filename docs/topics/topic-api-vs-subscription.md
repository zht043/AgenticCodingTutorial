## 1. 概述

Agent 工具有两种主流付费路径：API 按量付费（如 Claude Code 接 Anthropic API）和订阅制（如 Cursor Pro、Claude Max）。前者灵活但需要管理用量，后者简单但可能有额度上限。选择哪个取决于你的使用频率、任务复杂度和预算。
