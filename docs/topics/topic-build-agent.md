- Agent = Model + Harness 的工程实现
- 核心模块：LLM 接口、工具管理、上下文管理、循环控制
- 设计原则：最小权限、可观测、可中断

### 2. Claude Agent SDK 实战

- SDK 架构和核心 API
- 工具定义与注册
- 多轮对话管理
- 从零搭建一个代码审查 Agent

### 3. OpenAI Agents SDK 实战

- SDK 架构对比
- 工具调用与 function calling
- 多 Agent 编排
- Guardrails 与安全控制

### 4. 实战：构建专用 Agent

- 需求分析和架构设计
- 工具链选择
- 测试与调试
- 部署与运维
