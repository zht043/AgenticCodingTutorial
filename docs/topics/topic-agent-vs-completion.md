# 🤖 Agent vs 代码补全工具

> 🎯 理解 Agent 模式与传统代码补全（Copilot / Tab 补全）的本质区别，帮你在正确场景选择正确工具。

## 1. 概述

代码补全工具（如 GitHub Copilot Tab 补全、Cursor Tab）和 Agent 模式看似都在"帮你写代码"，但它们的运作机制和适用场景完全不同。补全工具是"你想写什么，我帮你写快一点"；Agent 是"你想达到什么目标，我帮你从头到尾做完"。理解这个区别，才能在对的场景用对的工具。
