## 1. 概述

Hooks 是 Agent 工具（特别是 Claude Code）提供的一种扩展机制，允许你在特定事件（如文件保存、工具调用前后）自动触发自定义脚本。它让你能在 Agent 的工作流中"插入"自己的逻辑，实现自动格式化、自动测试、自动提交等自动化流程。
