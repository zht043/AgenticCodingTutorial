## 1. 概述

与传统软件工程有设计模式一样，Agent 编程也有自己的"模式语言"。从最基础的 Prompt → Response 单轮模式，到 Plan → Execute → Verify 循环，再到 Orchestrator-Worker 多 Agent 模式——理解这些模式能让你在面对不同任务时快速选择合适的架构。
