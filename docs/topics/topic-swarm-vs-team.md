# 🐝 Agent Swarm vs Team

> 🎯 对比"群体智能"和"团队协作"两种多 Agent 组织模式的适用场景与权衡。

## 1. 概述

多 Agent 协作有两种典型范式：Swarm（松散的群体智能，每个 Agent 自主决策）和 Team（结构化团队，有明确的角色分工和协调者）。前者灵活但难以控制，后者可控但需要更多编排。理解二者的权衡，是设计多 Agent 系统的关键。
