> 🎯 从 CoT 到 Extended Thinking，理解 LLM 推理能力如何决定 Agent 的上限。

## 1. 概述

Agent 的核心引擎是 LLM，而 LLM 的推理能力直接决定了 Agent 能解决多复杂的问题。从 Chain-of-Thought 到 Extended Thinking（如 Claude 的 thinking mode），推理技术的进步让 Agent 从"补全代码"进化到"自主解决工程问题"。理解这些技术，你才能明白为什么同一个 Agent 在不同模型上表现差异巨大。
