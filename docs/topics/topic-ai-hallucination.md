## 1. 概述

"幻觉"是 LLM 最被人诟病也最需要正视的问题——模型会自信地编造不存在的 API、虚构函数签名、给出看似合理但完全错误的解释。在 Agent 编程中，幻觉不仅浪费时间，还可能引入隐蔽的 Bug。理解幻觉的成因和规律，是有效使用 Agent 的必修课。
