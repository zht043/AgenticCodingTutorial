## 1. 概述

现代 LLM 不仅能处理文本，还能理解图像、PDF、截图等多模态输入。在 Agent 编程场景中，这意味着你可以把 UI 设计稿、错误截图、架构图直接"喂给" Agent，让它基于视觉信息工作。这是一个经常被低估但极具实用价值的能力。
