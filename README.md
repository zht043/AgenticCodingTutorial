# Agentic Coding Tutorial

一份面向中文开发者的 Agentic Coding 教程规划仓库。

目标不是做一篇“工具罗列帖”，而是写成一套可以快速上手、逐步建立心智模型、最后真正能落地到日常开发中的教程。主线会以 `Claude Code` 为核心叙事工具，`Codex`、`Cursor`、`Cline`、`GitHub Copilot`、`Trae` 等作为对照与补充，覆盖模型选择、提示词方法、MCP、Skills、插件、权限、安全、成本控制和实战案例。

## 写作原则

- 先让读者“跑通一个例子”，再讲原理。
- 先区分“模型”和“Agent 产品”，再做横向比较。
- 先给出推荐方案，再谈可选方案和折中方案。
- 版本敏感内容单独维护，避免正文频繁过时。
- 每章都保留“适用场景”“避坑点”“可复制步骤”三个落点。
